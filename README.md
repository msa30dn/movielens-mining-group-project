# MovieLens Mining Group Project

A demonstration web application that shows how to apply **data mining techniques** to build a movie recommendation system using the [MovieLens](https://grouplens.org/datasets/movielens/) dataset.

---

## Overview

This project demonstrates how data mining can be applied to a recommendation function. Users can interact with a web interface to receive personalised movie recommendations generated by mining patterns from user rating history in the MovieLens dataset.

The application is intended as an educational tool to illustrate core data mining concepts such as:

- **Collaborative Filtering** – finding users with similar tastes and recommending movies they liked
- **Association Rule Mining** – discovering which movies tend to be watched or rated together
- **Clustering** – grouping users or movies by behaviour and preference patterns
- **Matrix Factorisation** – decomposing the user-movie rating matrix to uncover latent features

---

## Dataset

This project uses the [MovieLens dataset](https://grouplens.org/datasets/movielens/) provided by the [GroupLens Research Lab](https://grouplens.org/) at the University of Minnesota.

> F. Maxwell Harper and Joseph A. Konstan. 2015. The MovieLens Datasets: History and Context. ACM Transactions on Interactive Intelligent Systems (TiiS) 5, 4: 1–19. https://doi.org/10.1145/2827872

The dataset contains:
- Movie metadata (titles, genres)
- User ratings (1–5 stars)
- User-generated tags

---

## Features

- Browse and search movies
- Receive personalised movie recommendations
- View explanation of how mining was applied to produce each recommendation
- Compare recommendation strategies (collaborative filtering, association rules, etc.)

## Getting Started

### Prerequisites

- Python 3.9 or later
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/msa30dn/movielens-mining-group-project.git
cd movielens-mining-group-project

# Install dependencies
pip install -r requirements.txt
```

### Download the Dataset

Download the **ml-latest-small** dataset (recommended for development) from:
https://grouplens.org/datasets/movielens/latest/

Extract `ml-latest-small.zip` and copy the following CSV files into the `data/` directory at the project root (create the directory if it does not exist):

```
data/
├── movies.csv
├── ratings.csv
├── tags.csv
└── links.csv
```

### Run the Application

```bash
python app.py
```

Then open your browser and navigate to `http://localhost:5000`.
